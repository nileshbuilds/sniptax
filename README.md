**Sniptax**
Sniptax is a library of reusable web code snippets. Perfect for beginners, it lets you quickly add clean, modern sections to your websites.
Features
1. Ready-to-use HTML & CSS snippets
2. Categories: Navbars, Hero Sections, Feature Cards, Footers
3. Clean, beginner-friendly code
4. No external dependencies
How to Use
1. Open the snippets/ folder.
2. Pick a snippet you like.
3. Copy the HTML & CSS into your project.
4. Customize text, colors, and images.
Future Plans
1. Snippet-based website generator
2. Theme customization (light/dark/custom)
3. Live preview & one-click export
4. Contributing
5. Add new snippets, improve existing ones, or suggest features. Keep code clean and self-contained.
